Goxel

Open source voxels 3D editor for Mac, Linux and Windows.

Features

  • 24 bits RGB colors.
  • Unlimited scene size.
  • Unlimited undo buffer.
  • Multiple layers support.
  • Marching Cube rendering.
  • Procedural rendering.
  • Export to obj, pyl, png, qubicle.

Procedural Rendering

You can use code to generate complex 3d shapes.

The syntax is inspired from Context Free Art.

See the documention, or check the demo
